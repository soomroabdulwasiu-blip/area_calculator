#  Area Calculator

This repository contains a simple **C++ program** that calculates the **area of different shapes** using formulas and user input.

---

##  Features
- Calculates area of:
  - Circle
  - Rectangle
  - Triangle
- Uses `switch` statement for user choice
- Takes user input from console
